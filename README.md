# Profile-Based Matching Algorithm (Hybrid NLP & ML System)

## Description
This project implements a Profile-Based Matching Algorithm that utilizes advanced Natural Language Processing (NLP) techniques along with Machine Learning (ML) systems. It features an adaptive feedback mechanism to enhance match accuracy and relevance.

## Tech Stack
- Python
- spaCy
- NLTK
- Scikit-learn
- TF-IDF
- Cosine Similarity

## Key Contributions
- **Matching Algorithm:** Developed a robust algorithm to pair profiles effectively.
- **Feedback Loop:** Integrated a feedback loop for continuous improvement of matching processes.
- **Improved Match Acceptance:** Enhanced the acceptance rate of matches.

## Outcomes/Results
Achieved a significant improvement in matching accuracy, with enhancements ranging from 40% to 65%.

## Running Instructions
1. Clone the repository.
2. Install required packages using `pip install -r requirements.txt`.
3. Run the matching script with `python match_profiles.py`.
